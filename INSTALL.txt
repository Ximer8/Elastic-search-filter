🚀 QUICK INSTALLATION (5 MINUTES):

─────────────────────────────────────────────────────────────────────────────

Step 4: Test installation
─────────────────────────────────────────────────────────────────────────────

Step 5: Run first scan
─────────────────────────────────────────────────────────────────────────────

./scan.sh normal example_targets.csv

# Or with your own file
./scan.sh normal your_targets.csv

🐛 TROUBLESHOOTING:
═══════════════════════════════════════════════════════════════════════════════

Issue: "command not found: python3"
Solution:
  Ubuntu/Debian: sudo apt install python3
  macOS: brew install python3

Issue: "No module named 'requests'"
Solution:
  pip3 install requests --break-system-packages
  # Or: pip install requests
  # Or: sudo apt install python3-requests

Issue: "Permission denied"
Solution:
  chmod +x es_advanced_scanner.py
  chmod +x scan.sh
  # Or run with: python3 es_advanced_scanner.py

Issue: "No targets found in CSV"
Solution:
  - Check CSV format (see USAGE.txt)
  - Try different delimiter: --delimiter ";"
  - Verify file encoding: file -i targets.csv

Issue: Scripts are slow
Solution:
  - Increase workers: -w 100
  - Decrease timeout: -t 5
  - Use fast mode: ./scan.sh fast targets.csv

Issue: Too many timeouts
Solution:
  - Decrease workers: -w 10
  - Increase timeout: -t 20
  - Use thorough mode in pipeline

Issue: SSL/TLS errors
Solution:
  Scripts automatically ignore SSL verification (verify=False)
  This is intentional for security testing
